# Sales Forecasting System

A machine learning project to predict product demand using historical sales data.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Data preprocessing pipeline
- Feature engineering
- Linear regression model
- Model evaluation using MAE and RMSE
- Sales forecasting for future dates
- Visualisation of predicted vs actual sales

## Run Project

Install dependencies:

pip install -r requirements.txt

Run:

python main.py
